# Toaster-Library
A Simple library to show toast

How To Use

implementation 'com.github.Dev-Alaa:Toaster-Library:0.1.0'

Don't forget to add the JitPack repository to your build file

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
